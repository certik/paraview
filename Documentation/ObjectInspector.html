Object Inspector (View menu, Object Inspector)
The Object Inspector contains controls and information for the reader, source, or filter selected in the Pipeline Browser. There are three tabs in the Object Inspector: Properties, Display, and Information.

TabDescription
PropertiesControl parameters of the reader, source, or filter.
DisplayControl the appearance of the output of the reader, source, or filter.
InformationView information about the output of the reader, source, or filter.
